## 1 - Installation

1.1 - Clone the projet 
```
git@github.com:Cosmeak/CityBlock-V2.git
```

1.2 - Install dependencies

With yarn:
``` 
yarn install
```
or npm:
``` 
npm install
```

And install the expo cli:
- With yarn:
``` 
yarn global add expo-cli 
``` 

- With npm:
``` 
npm install -g expo-cli
```

1.3 - Launch the project

With yarn:
``` 
yarn start
```
or NPM:
``` 
npm start
```
